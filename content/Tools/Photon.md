#### Simple command
```shell
$ python3 photon.py -u google.com
```



```
$ python3 photon.py -u google.com -l 3 -t 200 --wayback
```

|           |                                                |
| --------- | ---------------------------------------------- |
| -u        | specifies url                                  |
| -l        | specifies levell to crawl                      |
| -t        | specifies number of threads (here,200)         |
| --wayback | specifies using URLs from archive.org as seeds |
